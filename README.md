Videogames-eshop-project
A full-stack Game Store application developed with Flask and Streamlit. Features include game management (CRUD), shopping carts, purchase tracking, game search and filtering, AI-generated summaries and recommendations via Groq, price comparison through web scraping, analytics dashboards, and SQLite/SQLAlchemy data persistence.
